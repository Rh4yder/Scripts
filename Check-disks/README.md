# PowerShell Scripts

Scripts úteis para administração de sistemas Windows.

## check-disks.ps1

Script em PowerShell para recolher informações de discos físicos e virtuais do sistema.

### Funções:
- Mostra o nome do host.
- Lista discos físicos: nome, estado, tamanho e fabricante.
- Lista discos virtuais: nome, estado e tamanho.

### Aviso:
> Este script é fornecido tal como está. Utilize com precaução e teste antes em ambientes de produção.
